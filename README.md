# GoldMine Sudan

**GoldMine Sudan** is an investment platform that connects investors with gold mining opportunities in Sudan. The platform provides tools for portfolio management, real-time market data, and secure transaction processing.

## Features

- **Investment Management** – Browse, invest in, and track gold mining projects
- **Portfolio Dashboard** – Monitor the performance of your investments in real time
- **Secure Transactions** – Safe and transparent payment processing
- **Market Data** – Up-to-date gold market prices and trends
- **User Authentication** – Secure login and account management

## Getting Started

### Prerequisites

Make sure you have the following installed:

- [Node.js](https://nodejs.org/) (v18 or higher)
- [npm](https://www.npmjs.com/) or [yarn](https://yarnpkg.com/)

### Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/fxuae277-cloud/goldmine-sudan.git
   cd goldmine-sudan
   ```

2. Install dependencies:
   ```bash
   npm install
   ```

3. Copy the environment configuration file and update the values:
   ```bash
   cp .env.example .env
   ```

4. Start the development server:
   ```bash
   npm run dev
   ```

## Usage

After starting the development server, open your browser and navigate to `http://localhost:3000` to access the platform.
